작문 (Writing)

AI 도구 구체 프롬프트 예시 활용 포인트
ChatGPT • 학습 목표에 맞는 영어 지문 3개 작성
• 학생 수준(HS2, 16세)에 맞춘 흥미로운 문장 구성
• 교재용 대화체 예문 생성		 • 주제와 수준을 명확히 제시하면 빠르게 초안 작성 가능
• 학습 목표, 단어 수준 명시 필수
Claude • 특정 문법 포인트를 활용한 예문 10개 생성
• 글의 톤: 재미있고 자연스럽게		 • 문체·톤 조절이 쉽고, ChatGPT와 비교해 장문의 글 작성에 강점
Jasper • AI 글쓰기 도구 활용, 학생 참여형 질문 포함 • 블로그나 SNS용 교육 자료, 시리즈 콘텐츠 제작 시 효율적

번역 (Translation)

AI 도구 구체 프롬프트 예시 활용 포인트
DeepL • 영어 지문을 한국어로 자연스럽게 번역
• 한국어 지문을 영어 학습용으로 변환		 • 번역 품질 높음
• 문맥·학습용 수준 고려 가능
ChatGPT + DeepL • ChatGPT로 영작 후 DeepL로 자연스러운 표현 개선 • 두 단계로 활용
• ChatGPT → 초안 생성 / DeepL → 자연스러운 문장
Google Translate • 간단 문장, 빠른 참고용 • 다소 직역 가능
• 참고용으로 활용

피드백 루프 체계

1

초기 검토

ChatGPT/Claude: 프롬프트를 구체화하고 초안을 받은 후, "이 내용을 더 쉽게", "다른 예시로" 피드백 제시

2

학년/수준 조정

초등학교: "이 문장을 더 쉽게", "단어를 더 간단하게" → ChatGPT 재생성

중학교: "이 내용을 조금 어렵게", "실생활 예시 추가"

고등학교: "논리적 흐름 강화", "심화 개념 포함"

3

검토 및 정제

문법: Grammarly로 문법, 스타일 체크

번역: DeepL로 자연스러운 문장 검증

출처: Perplexity로 정보 확인

4

최종 검증

교육과정 기준: 교육과정 기준에 맞는지 검증

난이도 확인: 학년별 적합성 확인

저작권 검토: 사용한 AI 도구의 저작권 정책 확인
