마케팅

SNS 캠페인, 광고 카피, 리서치

콘텐츠 기획 / 아이디어

AI 도구 구체 프롬프트 예시 활용 포인트
ChatGPT • 우리 출판사의 중등 수학 참고서의 SNS 캠페인 아이디어를 10가지 제안해줘
• 대상: 학부모(40대) 및 학생(13~16세)
• 목표: 브랜드 인지도 제고, '쉽게 배우는 수학' 이미지 강화
• 각 아이디어에 포함: ① 캠페인 제목 ② 핵심 메시지 ③ 추천 채널 ④ 콘텐츠 형식(영상/카드뉴스 등)		 • 타깃과 목표를 함께 명시해 '교육적 맥락'을 강조하면 아이디어가 실무에 가깝게 나옴
Gemini • 교육 콘텐츠 브랜드가 학부모 대상 홍보를 할 때, 신뢰와 전문성을 동시에 전달할 수 있는 콘텐츠 테마 5가지를 제안해줘. 각 테마에 맞는 블로그 주제 2개씩 추가해서 • Gemini는 검색 기반 자료를 활용하므로, '최신 교육 트렌드' 반영 아이디어 생성에 강점
Perplexity AI • 2025년 한국 교재 시장 마케팅 트렌드를 요약해줘. 특히 디지털 학습 도구, AI 학습 보조앱과 연계된 콘텐츠 마케팅 사례를 중심으로 • 시장 동향·경쟁사 캠페인 조사용으로 효율적
• 출처 기반 답변을 함께 제시함

SNS 콘텐츠

AI 도구 구체 프롬프트 예시 활용 포인트
ChatGPT • 인스타그램용 카드뉴스 문구를 작성해줘
• 주제: 개념 잡는 하루 10분 수학 루틴
• 톤: 따뜻하고 격려하는 교사 느낌 해시태그 8개 제안(#중등수학 #자기주도학습 등)		 • 구체적 문체 톤을 지정해야 감성 과잉/광고성 문구를 피할 수 있음
DALL·E 3 • 슬로건: 공부 습관은 반복에서 자란다
• 이 슬로건에 어울리는 교육 이미지(책상, 노트, 교과서)를 따뜻한 톤으로 생성		 • SNS 비주얼용 이미지를 생성해 글과 시각 요소의 일관성을 유지

광고 문안 / 캠페인 카피

AI 도구 구체 프롬프트 예시 활용 포인트
ChatGPT • 신간 참고서 광고 문구 3가지 버전 제안
1) 교사 추천 강조형, 2) 학습 효과 중심, 3) 감성(응원)형
• 각 버전마다 제목 + 부제 + CTA 포함		 • 톤별·목적별 카피 세트를 한 번에 만들 수 있어 A/B 테스트용으로 적합
Claude • 교육 플랫폼 광고 카피 작성
• 목표: 학습 신뢰감 + 과학적 학습법 강조
• 20자 이내 헤드라인 3개, 50자 이내 설명문 3개		 • Claude는 문체 일관성과 전문 어조를 잘 잡아 줌
Perplexity AI • 교육업계에서 성과가 높았던 온라인 광고 문구 사례 5가지 제시해줘
• 각 문구의 특징 분석도 포함해줘		 • 실제 성공 사례 기반으로 카피 개선 아이디어를 참고할 수 있음

피드백 루프 체계

1

초기 검토

ChatGPT: 카피 작성 후 "톤 조정", "CTA 강화" 피드백

2

학년/수준 조정

학부모 대상: "신뢰 톤", "전문성 강조", "교육적 가치"

학생 대상: "캐주얼 톤", "재미있고 흥미롭게"

교사 대상: "전문성 강조", "효과성 중심"

3

검토 및 정제

문체 통일: Claude로 일관된 톤 유지

경쟁사 분석: Perplexity로 시장 동향 확인

A/B 테스트: 여러 버전 비교 검증

4

최종 검증

지침 준수: 교육 커뮤니케이션 가이드 준수

플랫폼 규정: SNS/광고 플랫폼별 규정 확인

CTA 효과: 행동 유도 문구 검증
